
# Reversi

## 概要
本プログラムは Java で実装したReversi / オセロのアプリケーションです。

`Swing` を用いた GUI 上で動作し、マウス操作によってゲームを進行します。

人間プレイヤーだけでなく、複数の戦略を持つコンピュータプレイヤー（CP）を実装しており、各戦略の違いによる対戦結果や盤面評価の差を確認できます。

## 実行方法

### コンパイル
ソースコードのあるディレクトリで以下を実行してください。

```bash
javac Reversi.java
```

### 実行（プレイ）


```
java Reversi
```

このゲームにおいて、如何なる場合において、左クリックで黒の手を、右クリックで白の手を置くように設定されています。

### プレイヤー種別の指定（任意）

コマンドライン引数で黒・白それぞれのプレイヤータイプを指定できます。

```
java Reversi <Player1> <Player2>
```

| プレイ種別番号 | 意味                                       |
| :------------- | ------------------------------------------ |
| `0`            | リアルプレイヤー（人間によってマウス操作） |
| `1`            | コンピュータプレイヤー（戦略１）           |
| `2`            | コンピュータプレイヤー（戦略２）           |
| `3`            | コンピュータプレイヤー（戦略３）           |
| `4`            | コンピュータプレイヤー（戦略４）           |

例：

```
java Reversi 2 3
```

この場合、黒番は戦略２、白番は戦略３の CP が担当します。

```
java Reversi 0 4
```

この場合、黒番はリアルプレーヤー、白番は戦略４の CP が担当します。

## 戦略の説明

本プログラムでは、数値によって CP（Computer Player）の戦略を指定します。

- **1 : Random**

  置けるマスの中からランダムに 1 手を選択します。

- **2 : Greedy**

  その手で裏返せる石の数が最大となる位置を選択します（同点の場合はランダム）。

- **3 : Board Consideration**

  盤面の重み（角を高評価、角周辺を低評価など）と裏返し数を用いてスコア計算を行い、最も有利な手を選択します。

- **4 : Enhanced Board Consideration**

  戦略 3 を基に、次の相手の可動数（置ける手の数）を抑える評価を追加した戦略です。

  短期的な石数よりも、安定した盤面形成を重視します。

これらの戦略を組み合わせる、または人間プレイヤーと組み合わせることで、人対人・人対 CP・CP 同士の対戦が可能です。
